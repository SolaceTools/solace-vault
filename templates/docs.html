Overview

SOLiD Vault is a local-only, zero-knowledge encrypted vault for storing cryptocurrency seeds, private keys, and other sensitive data securely on your own device. No cloud storage is used.

Features

  • Local-only storage — no external servers or cloud required
  • Zero-knowledge encryption using Argon2id + AES-GCM
  • Automatic backup to a local JSON file for easy migration to other devices
  • Activity logs: track uploads, views, deletions, and login attempts
  • Flexible secret categorization (Seed, Private Key, Other) with labels and tags

Security

Every secret is encrypted with a unique AES key, which itself is encrypted with a key derived from your master password using Argon2id (memory-hard, state-of-the-art KDF).

Even if someone copies the backup file, without the correct master password the data is unreadable.

    Encryption Flow:

    Master Password ─► Argon2id ─► Master Key ─► Encrypt Secret Key ─► AES-GCM ─► Secret Data

Backup & Restore

  • Every upload automatically updates backup.json in your vault directory
  • You can import this JSON on another device as long as you use the same master password
  • Importing validates decryption before adding any secrets to the local vault

Usage

1. Setup a master password.

2. Upload seeds, private keys, or other sensitive information.

3. All operations (view, delete, change password) are logged automatically.

Activity Logs

The vault keeps detailed logs of actions:

  • Login attempts (success and failure)
  • Secret uploads, views, and deletions
  • Password changes
  • Backup imports

Logs can be viewed in the "Logs" section of the app for auditing and security purposes.
